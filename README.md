# 智能安防巡检机器人 | 大学生大创项目
## 项目简介
本项目针对实验室人工巡检效率低、风险高、单一传感器感知不足等痛点，基于 ROS2 Foxy 框架开发全自动智能安防巡检机器人。
系统集成多类技术：使用 GMapping 算法完成激光 SLAM 建图、AMCL 粒子滤波实现高精度定位；基于 DWA 动态窗口法完成实时动态避障；通过 YOLOv5 训练火灾检测模型，精准识别火焰、烟雾，实现火灾隐患实时预警，可落地用于实验室常态化安防巡检。

### 核心功能
1. 实验室场景自主建图、厘米级实时定位
2. Nav2 全局路径规划 + DWA 动态障碍物避障
3. 火焰、烟雾双目标智能识别与火灾预警
4. 全流程无人值守全自动安防巡检

## 开发与运行环境
- 操作系统：Ubuntu 20.04
- 核心框架：ROS2 Foxy
- 核心算法：GMapping SLAM、AMCL 定位、DWA 动态避障、YOLOv5 目标检测
- 硬件设备：激光雷达、深度相机、机器人运动底盘

## 简易运行步骤
1. 部署 Ubuntu 20.04 + ROS2 Foxy 环境，配置清华软件源并安装全部依赖
2. 接入激光雷达、深度相机与机器人底盘驱动
3. 启动 SLAM 建图程序，采集并保存实验室栅格地图
4. 加载地图，开启 AMCL 定位与 Nav2 导航、DWA 避障模块
5. 运行 YOLOv5 视觉检测程序，实时监测火焰、烟雾隐患
